# Agent Instructions

- Do not make code changes the user did not explicitly ask for.
- Keep edits tightly scoped to the requested files and behavior.
- If a related cleanup, refactor, test change, or dependency change seems useful, ask first instead of applying it.
- Do not assume a broader implementation plan is wanted when the user asks a narrow question or targeted change.
- Do not create fake tests, mocked tests, or replacement tests unless the user explicitly asks for them.
